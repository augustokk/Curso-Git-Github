GIT

Git eh um software de controle de versao ele serve para voce manter seus repositorios de forma local

A ideia de usar o git eh a possibilidade de versionamento, com isso voce evitaria de ficar criando diversos folders, por exemplo, trabalho final. doc / trabalho.final2 / trabalh.agoravai ...

Usando o Git somos capazes de commitar o nosso trabalho para um repositorio central

As principais vantagens sao:

  • Controle de historico
  • Facilidade em trabalho em equipe
  • Possibilidade da ramificacao do projeto
  • Seguranca
  • Organizacao

    • GITHUB

    Eh uma plataforma de rede social para programadores

    Usando o GitHub somos capazes de "pushar"" o nosso trabalho para um repositorio remoto

    As principais vantagens sao:

  • Repositorios ilimitados
  • Hospedagem de codigo fonte
  • Caracteristica de rede social
  • Colaboracao
  • Forks
